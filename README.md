# YouTube Subtitle Extractor

Extract timestamped subtitles from YouTube videos and export them as TXT or SRT.

## Features

- Extract subtitles
- Timestamp formatting
- TXT Export
- SRT Export
- Responsive UI
- Rate limiting
- Helmet security
- Production ready

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production

```bash
npm start
```

Visit

```
http://localhost:3000
```

## Tech Stack

- Express 5
- TailwindCSS
- YouTube Transcript API
- Helmet
- Express Rate Limit
- Dotenv
